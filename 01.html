Create a Rails app

Start with a brand new Rails app and give it a minimal setup:

 
  1. Create a Rails repo calling rails new marketplace with a few options: --main to point to the most recent version of Rails, -d postgresql to use PostgreSQL as the database, and --skip-solid to skip Solid (not used by this project).
  2. Clean up the Gemfile to only include required libraries at their most recent version.
  3. Replace ' 3000' with ' 2026' in the code so we can run the server locally at a fancy http://localhost:2026 URL.
  4. Tell Rails not to generate unit tests for new models (spoiler: you‚Äôll use system tests instead).
