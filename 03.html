We add comments to classes and methods and let Yard generate documentation for the codebase:

  1. ▶︎ Add yard to the Gemfile (and doc/ to .gitignore).
  2. ▶︎ Add yard-style comments on top of every public method and class.
  3. ▶︎ Add a check in bin/ci to fail unless the code is 100% documented.
  4. ▶︎ Invoke `bin/yard && open doc/index.html` to display the generated docs.
