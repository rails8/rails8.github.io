We build the first two pages—a form to collect the homeowner's phone number and its confirmation:

  1. ▶︎ Create a Booking model (with a required phone) calling: rails g model booking phone:string!:index.
  2. ▶︎ Create a Contacts controller with two actions (new, create) and an HTML form to submit a phone number.
  3. ▶︎ Create a Confirmations controller with one action (show) and an HTML view to display the confirmation.
  4. ▶︎ Improve the design extracting common HTML and embedding Bootstrap in the Application layout.
  5. ▶︎ Create a Stimulus controller to add separators to the phone number while it’s being typed.
  6. ▶︎ Adjust the routes to display the pages at /contact and /confirmation.
