We use the Twilio API to send messages and confirm they were delivered:

  1. ▶︎ Create a Notification model with: rails g model notification booking:references delivered_at:datetime.
  2. ▶︎ Create a Notified concern to generate a notification whenever a booking is created.
  3. ▶︎ Create a Delivery model with: rails g model notification:references sid code.
  4. ▶︎ Create a Stated concern to manage the delivery status with PostgreSQL enums.
  5. ▶︎ Create a Deliverable concern with one actions (show) and an HTML view to display the confirmation.
  6. ▶︎ Improve the design extracting common HTML and embedding Bootstrap in the Application layout.
  7. ▶︎ Create a Stimulus controller to add separators to the phone number while it’s being typed.
  8. ▶︎ Adjust the routes to display the pages at /contact and /confirmation.
